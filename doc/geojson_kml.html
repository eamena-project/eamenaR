geojson_kmlR Documentation

Convert from/to GeoJSON, KMZ or KML

Description

Convert from KML/KMZ to GeoJSON, or from GeoJSON to KML. Geometries drawn in Google Earth or Google Earth pro can be exported as KML or KMZ (ie, compressed KML). At the same time, geometries drawn in EAMENA can be exported as GeoJSON. For a given HP, this data can be then imported into the EAMENA DB.

Usage

geojson_kml(
  geom.path = paste0(system.file(package = "eamenaR"), "/extdata/Waypoints.kmz"),
  geom.types = c("POINT", "LINE", "POLYGON"),
  export = T,
  dirOut = paste0(system.file(package = "eamenaR"), "/extdata/"),
  geojson.name = "Waypoints",
  selectedName = "EAMENA.ID",
  selectedFields = c("EAMENA.ID", "Resource.Name", "resourceid"),
  verbose = T
)

Arguments

geom.path

the path to the KML, KMZ or GeoJSON file.

geom.types

the types of geometries ("POINT", "LINE" or "POLYGON") that will be selected. By default all: 'c("POINT", "LINE", "POLYGON")'.

export

if TRUE, will export the KML/KMZ file in a GeoJSON format, or the GeoJSON file as a KML, if FALSE simple plot.

dirOut

the path to the folder where the KML/KMZ/GeoJSON file will be created.

geojson.name

the name of the KML/KMZ/GeoJSON that will be created without the extension

selectedName

for KML export only. The field selected to be the KML name of the HP, by default "EAMENA.ID".

selectedFields

for KML export only. KML conversion remove a large number of GeoJSON fields. This variable is used to select the fields we want to preserve. By default: c("EAMENA.ID","Resource.Name", "resourceid").

verbose

if TRUE (by default) then display different messages.

Value

a GeoJSON file or a KML file

Examples


library(dplyr)

# from KMZ to GeoJSON
library(dplyr)
geojson_kml(geom.types = "POINT")

# from GeoJSON to KML
geojson_kml(geom.path = "C:/Rprojects/eamenaR/inst/extdata/kites.geojson",
                   dirOut = "C:/Rprojects/eamenaR/inst/extdata/",
                   export = T,
                   geojson.name = "kites_outKML")
